    Virtualización

    La virtualización permite ejecutar varios sistemas operativos sobre un mismo equipo físico mediante el uso de máquinas virtuales administradas por un hipervisor. Esta tecnología optimiza el uso del hardware y facilita el desarrollo, las pruebas y la administración de sistemas.

    💻

    Máquina Virtual

    Simula una computadora completa mediante software.

    ⚙️

    Hipervisor

    Software encargado de crear y administrar las máquinas virtuales.

    🖥️

    VirtualBox / VMware

    Herramientas ampliamente utilizadas para virtualizar sistemas operativos.

    🚀

    Beneficios

    Mayor aprovechamiento del hardware, aislamiento y facilidad para realizar pruebas.

    Computación en la Nube

    La computación en la nube permite almacenar información, ejecutar aplicaciones y acceder a recursos informáticos mediante Internet, sin depender exclusivamente del equipo local.

    IaaS

    Infraestructura como servicio. Proporciona servidores, almacenamiento y redes virtuales.

    PaaS

    Plataforma como servicio. Facilita el desarrollo y despliegue de aplicaciones.

    SaaS

    Software como servicio. Aplicaciones listas para utilizar desde un navegador web.

    Google Drive

    Ejemplo de servicio SaaS utilizado para compartir y organizar el proyecto.
